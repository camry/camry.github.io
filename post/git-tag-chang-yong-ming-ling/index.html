Git tag 常用命令

创建 tag

注解：git tag 是打标签的命令，-a 是添加标签，其后要跟新标签号，-m 及后面的字符串是对该标签的注释。

git tag -a v1.0.1 -m "Relase version 1.0.1"

提交 tag 到远程仓库

git push origin -tags

删除 tag

git tag -d v1.0.1

删除远程 tag

git push origin :refs/tags/v1.0.1

查看 tag

git tag
git tag -l
