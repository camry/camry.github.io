Git branch 常用命令

2021-12-01 1 min read

Branch 常用命令

打包

差异文件打包

git diff <commit> <commit> --name-only | xargs zip v1.0.0.zip

分支

查看已经合并进来的分支

git branch --merged

查看尚未合并进来的分支

git branch --no-merged

推送本地分支

git push origin 分支名称
git fetch origin

跟踪远程分支

git checkout --track origin/serverfix

删除远程分支

git push origin :serverfix

分支的衍合

# git rebase --onto 基底分支 特性分支...
git rebase --onto master server client
# 将client分支合并至master
git checkout master
git merge client
# git rebase 主分支 特性分支
git rebase master server
# 将server分支合并至master
git checkout master
git merge server
# 删除client分支
git branch -d client
# 删除server分支
git branch -d server

注：把一个分支中的修改整合到另一个分支的办法有两种：merge 和 rebase（译注：rebase 的翻译暂定为“衍合”）。
